Features

Multi-Monitor Capture

Capture all connected displays as a single canvas, then save individual monitors or all at once.

Global Hotkeys

Customizable hotkeys to save replays instantly. Assign direct hotkeys per monitor for quick saves.

NVENC Encoding

Hardware-accelerated encoding with minimal CPU/GPU impact while maintaining quality.

Auto Cropping

Automatic per-monitor splitting using FFmpeg. Save exact displays without manual editing.

Configurable

Control buffer length, bitrate, FPS, resolution, audio sources, encoder, and output format.

System Tray

Runs quietly in your system tray with quick access to controls and settings.

Default Hotkeys

All hotkeys are customizable in settings.
